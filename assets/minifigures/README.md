# Minifigures — Generated Concept Art

AI-generated character concept art for tabletop miniatures, organized one subfolder per figure.

Each minifigure gets its own folder under `assets/minifigures/<slug>/` with:

- PNG renders (front views, turnaround sheets, revisions)
- A `README.md` cataloging each image, print notes, and GitHub direct links for phone viewing

## Adding a new minifigure

1. Create `assets/minifigures/<kebab-case-slug>/`
2. Save images as `<slug>-v1.png`, `<slug>-v2.png`, etc.
3. Add a `README.md` with an image table and direct GitHub links
4. Update the table in this file
