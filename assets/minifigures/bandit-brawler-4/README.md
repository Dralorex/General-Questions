# Bandit Brawler 4 — Minifigure Concept

Generated unpainted gray resin tabletop miniature concept. 32mm heroic scale, 25mm round base, 25µm layer-friendly geometry with structural connections.

Concept: bare-knuckle fighting pose, brass knuckles over main fist knuckle joints (MCP), sleeveless jerkin, scars and tattoos, bandana over nose.
