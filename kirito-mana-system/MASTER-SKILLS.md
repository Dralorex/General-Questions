# Master Skills Sheet

One-page GitHub view of the Kirito Aincrad mana kit.

**Max Mana:** 100 · **Flow:** Level unlocks → train sessions → Learned → usable  
**Dual wield:** locked until **Level 3** (Reaction Trial + Dual Blades training)  
**No projectile skills.** Melee / movement / defense only.

→ Rules: [README.md](README.md) · **Training guide (drills & hours):** [TRAINING.md](TRAINING.md) · D&D sheet: [DND-SHEET.md](DND-SHEET.md) · App: [../kirito-mana-app/](../kirito-mana-app/) · CSV: [skills.csv](skills.csv)

## Dual wield gate (Level 3)

1. Level **3+**  
2. **Uppercut** Learned  
3. **Reaction Trial** completed  
4. **4 sessions** (~6 hr) on Dual Blades Unique Skill  

Then: Double Circular → Cross Block → … → Starburst → Prelude → Eclipse
